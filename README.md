## Overview
Sea Shark is a C# console application demonstrating object-oriented programming (OOP) principles, including:
- **Encapsulation**: Private fields controlled by public properties and methods.
- **Inheritance**: `BeginnerLevel` and `AdvancedLevel` inheriting from a base `Level` class.
- **Polymorphism**: Overriding the abstract `LoadQuiz` method to provide a level-specific implementations.
- **Abstraction**: Using abstract class to define common attributes and methods.

## Features
- Player's movement tracking.
- Multiple levels of difficulty (Beginner and Advanced).
- Time tracking with countdown logic.
- Interactive quiz system, determining score by validating user input.

## How to Run
1. Open a terminal or command prompt.
2. Navigate to the project directory.
3. Run the project using the .NET CLI:
   ```bash
   dotnet build
   dotnet run
   ```
4. Follow the on-screen prompts to play the SeaShark game!
